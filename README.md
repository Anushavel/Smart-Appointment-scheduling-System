# Smart-Appointment-scheduling-System

A full-stack web application to book and manage appointments efficiently. Built using HTML, CSS, JavaScript, Spring Boot, and MySQL with REST APIs and CRUD operations.

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Create database: CREATE DATABASE appointment_db;
2. Update MySQL username and password in application.properties.
3. Run the Spring Boot application.
4. Open: http://localhost:8080/index.html
